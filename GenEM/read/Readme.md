# 📘 Genetic Predisposition Analysis Panel System – Three‑Level Knowledge Framework
This program is a fully offline genetic analysis system designed to interpret various predispositions and traits from **MyHeritage RAW DNA files**.

Its goal is to present genetic patterns in a clear, understandable way, showing how they may influence:


- metabolism  
- physical characteristics  
- lifestyle tendencies  
- and overall biological functioning  

The system is built on a **three‑level knowledge framework**, combining scientific literature, phenotype correlations, and practical interpretation layers.
 In memoriam Orczán Csaba Sándor (1948-2026)

---

## 🔬 Supported Input Format

- **MyHeritage RAW DNA file**  
- Fully offline processing  
- No external API calls  
- No internet connection required  

---

## 🧠 Three‑Level Knowledge Framework

### **Level 1 – Genetic Marker Interpretation**
Each SNP is evaluated based on:
- allele impact  
- known phenotype associations  
- metabolic or structural relevance  

### **Level 2 – Pattern Recognition**
The system identifies:
- metabolic tendencies  
- physical predispositions  
- lifestyle‑related patterns  
- combined SNP effects  

### **Level 3 – Human‑Readable Summary**
The final output provides:
- clear explanations  
- practical interpretations  
- easy‑to‑understand summaries

## 🛠 Installation

Installing GenEM on Windows is simple and straightforward.

### 1. Download the program
Download the GenEM project from GitHub:

1. Open the project page.
2. Click **Code → Download ZIP**.
3. Extract the ZIP file.

Rename the extracted folder to **GenEM** and place it on your **Desktop**.  
This is the easiest way to keep the program accessible at all times.

### 2. Install Python
GenEM requires Python 3.10 or newer.  
Download: https://www.python.org/downloads/

Make sure to enable pip during installation.

### 3. Launching GenEM on Windows

GenEM can be launched in two ways:

#### 1) indit.bat – automatic desktop shortcut creation
The `indit.bat` file performs the following actions:
- starts the GenEM program,
- creates a desktop shortcut,
- allows launching GenEM later with a single click from the desktop icon.

Usage:
1. Open the **GenEM** folder on your Desktop.
2. Double-click `indit.bat`.
3. The program starts.
4. A new desktop icon appears: **GenEM**.
5. From now on, you can launch GenEM with one click.

#### 2) genem.bat – direct launch from the folder
You can also start GenEM directly by clicking the `genem.bat` file.

Usage:
1. Open the **GenEM** folder on your Desktop.
2. Double-click `genem.bat`.
3. The program starts immediately.

### 4. Using your personal DNA file
Place your personal **_raw_dna_data.csv_** file inside the GenEM folder, as the program reads this file for analysis.  
All panel evaluation results are automatically saved as **.txt files** inside the GenEM folder.

### 5. Program language
GenEM currently runs **in Hungarian**.  
If you need the results in English, please use a **Hungarian–English translator**.  
An English version of the program is under development.

## ⚠️ Data Security Notice

GenEM processes extremely sensitive personal information, as RAW DNA files contain genetic data.  
The program works **entirely offline**, does not send any data to servers, and does not connect to the internet.

Please pay special attention to:
- keeping your _raw_dna_data.csv_ file private,
- storing it in a secure location,
- not sharing it with others,
- running GenEM only on your own computer.

The .txt result files generated by GenEM also contain personal information, so handle them carefully.
